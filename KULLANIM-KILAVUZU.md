# Kumbara Takip Sistemi - Kullanım Kılavuzu

## 💻 Localde Çalıştırma

### 🚀 Hızlı Başlatma

`uygulamayi-baslat.bat` dosyasına **çift tıklayın**!

Bu dosya:
1. Otomatik olarak `web` klasörüne gider
2. Node modülleri yoksa `npm install` çalıştırır
3. Geliştirme sunucusunu başlatır
4. Tarayıcınızda http://localhost:3000 adresine gidin

### ⏹️ Durdurma

Terminalden **Ctrl + C** tuşlarına basın.

## 📁 Dosyalar

- `uygulamayi-baslat.bat` - Uygulamayı başlatır
- `uygulamayi-durdur.bat` - Durdurma bilgisi (isteğe bağlı)
- `web/src/app/page.tsx` - Ana sayfa dosyası
- `web/src/components/` - Bileşenler klasörü

## 🎯 Özellikler

1. 🗺️ Harita üzerinde konum seçerek firma ekleme
2. 📋 Firma listesi görünümü
3. 🔍 Firma arama
4. 📍 Harita üzerinde gezinme
